# otel-demo

OpenTelemetry auto-instrumentation demo.

## Build

```
$ make
```

## Run

```
$ docker-compose up
```

## Endpoints

- Sample app:
  - UI: http://localhost:8080
  - API: http://localhost:8081
  - Provider 1: http://localhost:8082
  - Provider 2: http://localhost:8083
- Jaeger UI: http://localhost:16686
